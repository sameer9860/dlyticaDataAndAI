# Dlytica — Data & AI Course

A compact repository for Dlytica course materials: focused exercises and examples to learn Python, data processing, and PostgreSQL.

What you'll find
- Short, self-contained Python examples and scripts.
- PostgreSQL examples and SQL snippets for learning and practice.
- Small notes and instructions alongside code (open files to view details).

How to use
- Clone the repository and open the Python folder:

```bash
git clone https://github.com/sameer9860/dlyticaDataAndAI.git
cd dlyticaDataAndAI
```

- Run Python examples locally:

```bash
python3 dlyticaDay1.py  [# for linux] and 
python dlyticaDay1.py   [# for windows]
```

Conventions
- Keep examples small and focused; name files clearly.

Goal
- Practical, focused practice to build skills in Python, data handling, and PostgreSQL.
